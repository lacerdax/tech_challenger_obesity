# Tech Challenge - Predição de Obesidade (Fase 04)

Este projeto treina um modelo de Machine Learning para prever o nível de obesidade (multiclasse) e disponibiliza uma aplicação preditiva via Streamlit.

## Estrutura

- `data/Obesity.csv` → dataset
- `src/train.py` → treino + pipeline + export do modelo
- `src/app.py` → app Streamlit
- `models/` → modelo salvo e métricas

## Como rodar localmente

### 1) Instalar dependências

```bash
pip install -r requirements.txt
```
